What is Support Hub

Support Hub is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. You can interact securely with any Dapp from their mobile phone, making Support Hub wallets a safer choice compared to desktop or browser extension wallets.

How does it work?

Support Hub connects web applications to Support Hubed mobile wallets. Hubs One session is started by a scanning a QR code (desktop) or by clicking an application deep link (mobile).
